# 控制安全建模分析法

## 核心公式

缺失类漏洞 = 敏感操作 - 应有控制

敏感操作存在，但应有的安全控制缺失 = 漏洞

## 安全控制建模&控制存在性验证

### 敏感操作分类

| 类别         | 识别特征                             | 典型示例              | 风险等级 |
| ------------ | ------------------------------------ | --------------------- | -------- |
| **数据修改** | POST/PUT/DELETE                      | 创建/修改/删除        | 高       |
| **数据访问** | GET + ID参数                         | /user/{id} /用户/{id} | 中       |
| **批量操作** | export/download/batch 导出/下载/批量 | 导出/批量删除         | 高       |
| **权限变更** | role/permission/grant 角色/权限/授予 | 角色/权限分配         | 严重     |
| **资金操作** | transfer/pay/refund 转账/付款/退款   | 转账/支付/退款        | 严重     |
| **认证操作** | login/password/token 登录/密码/令牌  | 登录/密码重置         | 严重     |

### 安全控制矩阵

| 敏感操作类型 | 必须控制                         | 可选控制 |
| ------------ | -------------------------------- | -------- |
| **数据修改** | 认证、授权、资源所有权、输入验证 | 审计日志 |
| **数据访问** | 认证、资源所有权/范围限制        | 数据脱敏 |
| **批量操作** | 认证、授权、数量限制             | 频率限制 |
| **权限变更** | 认证、高级授权、权限边界检查     | 审批流程 |
| **资金操作** | 认证、授权、幂等性、并发控制     | 风控检查 |

### 缺失控制 → 漏洞映射

| 缺失控制       | 漏洞类型       | CWE     |
| -------------- | -------------- | ------- |
| 认证缺失       | 认证绕过       | CWE-306 |
| 授权缺失       | 功能级授权缺失 | CWE-862 |
| 资源所有权缺失 | IDOR           | CWE-639 |
| 授权逻辑错误   | 授权错误       | CWE-863 |
| 幂等性缺失     | 重放攻击       | CWE-294 |
| 并发控制缺失   | 竞态条件       | CWE-362 |
